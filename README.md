# BeNext 

This app adds support for BeNext devices in Homey.
It is a rewrite from to SDKv2
Not all features are implemented at the moment. Please report any problems and suggestions.

Changelog
V2.0.0
Rewrite to SDKv2

## Currently supported devices:

* Alarm sound
* P1 dongle
* Tag Reader 500
* Plug-in dimmer
* Built-in dimmer
* MoLiTe sensor
* Window/door sensor
* Heating control
* Energy switch plus
